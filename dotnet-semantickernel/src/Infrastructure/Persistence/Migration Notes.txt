1.  Use PM>
2.  Set Default Project to Aacn.SemanticKernel.Infrastructure
3.  Invoke Migration Commands

Add Migration
add-migration init -context SemanticKernel.Infrastructure.Persistence.ForecastsContext -verbose -s Presentation.WebApi -OutputDir Persistence/Migrations
add-migration ForecastsView -context SemanticKernel.Infrastructure.Persistence.ForecastsContext -verbose -s Presentation.WebApi -OutputDir Persistence/Migrations

Rollback Migration
remove-migration -context SemanticKernel.Infrastructure.Persistence.SemanticKernelContext  -s Presentation.WebApi –force

Update Database
update-database -context SemanticKernel.Infrastructure.Persistence.ForecastsContext  -s Presentation.WebApi

Reverse engineer view example
